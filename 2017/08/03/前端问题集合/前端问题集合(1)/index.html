前端问题整理集合(1)

发表于 2017-08-03

导语

出来工作一年多了，接触过许多前端工具相关的知识，在实际的开发中也遇到过许多问题，问题有大有小，每个常见的问题我都把他记录下来，
以方便以后忘记的时候可以拿出来预习下，巩固下自己的专业知识。

Jquert
  • 问题: 由bind绑定的事件是无法针对后期动态生成的dom实现
    解决: 使用on或者live进行DOM事件的绑定
    angualr
  • 问题: 组件通讯(rxjs 流)
    解决:import { Subject } from “rxjs/Subject”;//引入Subject包，如没有则需要在第三方库中引入
  • 问题:Property ‘entries’ does not exist on type ‘ObjectConstructor’显示没定义
    解决:把object换成(Object)
  • 问题: 环境安装（angular2 cli 环境）
    解决: npm install -g typescript typings npm install -g @angular/cli
  • 问题:如何动态嵌入组件
    解决:首先引入当前需要生成的组件，然后在需要嵌入组件的HTML模板里声明引用变量
npm
  • 问题:Cannot find module ‘包名称’
    解决:要么安装的路径错了，要么就是没装,重装一次.
  • 问题: npm istall 超时
    解决:
    • 要么找其他已安装好的项目依赖包，要么通过换URL渠道安装包
    • 改git代理 npm config set proxy http://127.0.0.1:1080
    • 在install附加参数 –registry=https://registry.npm.taobao.org
  • 问题: Cannot find module ‘copy-webpack-plugin’
    解决: npm install copy-webpack-plugin
  • 问题: Error: %1 is not a valid Win32 application.
    解决: npm install node-sass
